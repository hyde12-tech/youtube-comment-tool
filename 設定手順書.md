# YouTube コメント取得ツール 設定手順書

このツールを使用するには、事前に2つの準備が必要です。

---

## 事前準備①：YouTube APIキーの取得

### 1. Google Cloud Console にアクセスする
https://console.cloud.google.com/ を開き、Googleアカウントでログインします。

### 2. プロジェクトを作成する
1. 画面上部の「プロジェクトを選択」をクリック
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を入力（例: `youtube-comment-tool`）して「作成」

### 3. YouTube Data API v3 を有効にする
1. 左メニュー「APIとサービス」→「ライブラリ」
2. 検索欄に「YouTube Data API v3」と入力
3. 「YouTube Data API v3」をクリック → 「有効にする」

### 4. APIキーを取得する
1. 左メニュー「APIとサービス」→「認証情報」
2. 「認証情報を作成」→「APIキー」をクリック
3. 表示されたAPIキーをコピーしておく

### 5. .envファイルにAPIキーを設定する
ツールフォルダ内の `.env` ファイルをメモ帳で開き、以下のように書き換える:

```
YOUTUBE_API_KEY=AIzaSy...（コピーしたAPIキー）
```

---

## 事前準備②：Google認証ファイルの取得

### 1. OAuth同意画面を設定する
1. 左メニュー「APIとサービス」→「OAuth同意画面」
2. ユーザーの種類：「外部」を選択 → 「作成」
3. アプリ名（例: `youtube-comment-tool`）と自分のメールアドレスを入力 → 「保存して次へ」
4. スコープは変更せず → 「保存して次へ」
5. テストユーザーに自分のGmailアドレスを追加 → 「保存して次へ」

### 2. 認証情報（credentials.json）を取得する
1. 「認証情報」→「認証情報を作成」→「OAuthクライアントID」
2. アプリケーションの種類：「デスクトップアプリ」を選択
3. 名前を入力（何でもOK）→ 「作成」
4. 「JSONをダウンロード」をクリック
5. ダウンロードしたファイルを **`credentials.json`** にリネームして、.exeと同じフォルダに置く

## ツールの使い方

### フォルダ構成の確認
以下のファイルが揃っていることを確認してください:

```
📁 配布フォルダ/
├── ✅ youtube_comment_tool.exe
├── ✅ .env（APIキー設定済み）
└── ✅ credentials.json
```

### 起動と操作
1. `youtube_comment_tool.exe` をダブルクリック
2. 「YouTube URL」欄に動画のURLを貼り付ける
3. スプレッドシートURLは省略（Enterキー）か、既存のURLを入力
4. 「取得開始」ボタンをクリック
5. **初回のみ** ブラウザが開くのでGoogleアカウントにログイン
6. 完了後、「スプレッドシートを開く」ボタンをクリック

## よくあるエラーと対処

| エラーメッセージ | 対処法 |
|--------------|--------|
| `.envファイルにYOUTUBE_API_KEYが設定されていません` | `.env`ファイルを開いてAPIキーを正しく貼り付けてください |
| `credentials.jsonが見つかりません` | `credentials.json`が.exeと同じフォルダにあるか確認してください |
| `本日のAPI使用量の上限に達しました` | YouTube APIの無料枠（1日10,000クォータ）を超えました。翌日再試行してください |
| `この動画はコメントが無効です` | その動画のコメント欄が閉じられています。別の動画で試してください |
